Book#01：了不起的我（自我发展的心理学）

Book#01：了不起的我（自我发展的心理学）陈海贤

今天偶然看到有人推荐这本书，就顺手下载并翻阅了一下。今年给自己的阅读计划是，从读过的每本书上吸收三点内容，实践一项内容。开始吧

三个受益点：

  1. 很喜欢这本书里一个比喻，来帮助我们形象地理解为什么改变很难。

    人的情感面就像一头大象，而理智就像一个骑象人，骑象人其在大象背上，手里握着缰绳，好像是他在指挥大象，但事实上，他的力量微不足道，一旦和大象发生冲突，他想往左，而大象想往右，那他通常是坳不过大象的。

  2. 消费型快乐 Vs. 创造型快乐

    生活的乐趣分为2种，消费型快乐和创造型快乐。

    在消费型快乐，你消费别人创造的产品，满足的是表面上的感官刺激。

    在创造性快乐里，你在创造自己的产品，你在发挥自己的才能，幸苦地工作，在这个过程中，你体会到一种深刻的成就感，一种自己正在变得更好的感觉。

    如果说消费型快乐是酒肉朋友，那么创造型快乐就是良师益友。

  3. 创造型思维 Vs. 解决问题思维

    创造型思维是以目标来思考现实，先想我要什么，再想现实是怎样的，环境能够提供什么，再去想办法弥补目标和现实之间的鸿沟。

    解决问题思维是以现实来思考目标。先去想环境能够提供什么，再来想自己的目标是不是实际，该树立怎么样的目标。

    只有创造型思维才能产生持续行动的张力。

1个实践项：

努力控制你所能控制的事情，并接纳你不能控制的事情。（古希腊斯多葛学派的主张）

这个说法有听过，在这里被得到重复，外加最近发生的几件事情，尤有感触。未来凡事记得先区分事情是自己可控还是不可控的，可控就努力/尽力去做，不可控接纳它就好，不再花精力在它上面。

新年第一篇weekly report

今天周五，写了新年第一篇weekly report。和去年不同的是，因为有每日工作日志的帮助，今天写起来特别容易。

工作日志原计划是每天工作结束用10分钟左右做总结，后来在第一个工作日就被我改为边工作边做记录。每完成一项工作就马上记录下来，每收获一个知识点，也马上记录下来，每一个bug的solution也记下来。这样真的可以做到清楚地回顾，预览自己每一天是如何度过的，或者说是如何进步的。

这种有意识地活着的感觉好极了。

小调整

每天的一小时阅读计划，在开始读专业书后就无心读非专业书了，而专业书都是上千页，读完消化需要时日，这个问题困扰自己好几天了。刚在跑步机上突然有了个小的想法：不如把专业书调整到公司去读，每天早一点到公司，利用刚到公司的一小段时间，亦或是效率低下的工作间隙。在公司读更方便实践，边读边实践，或边读边优化自己的代码。恩，明天开始先尝试做这个小调整，看看会不会引发其它问题。

旁观自己

今天在试图旁观自己，旁观自己是如何拖延的。

早上睁眼，试图不看手机，直接下楼，但还是失败了，只是看的内容稍稍收敛了点，时间稍微短了点。

在公司，效率一如既往地低，一方面思维不够敏捷，其次无法做到deep work。唯一改进地是，实时更新了工作日志，尝试了在线共享文档的方式。

晚上回到家，处理完各种事情，坐到书桌前，还是迟迟无法开始阅读，当然最终还是完成了新年第一个工作日的一小时阅读计划。只是很明显，如果不刻意地，反复地提醒自己做这件事的话，这件事是一定不会发生的。

跳出自我，观察自己，最大的感受是：一个人改变自己很难，不存在什么新年新气象，或者至少不存在新年自动新气象，需要每日点滴地提高自己，改变自己，然后数日数月之后才有可能看到一点新气象。

跨年

除了做了顿大餐，今年的跨年和其它日子别无二致。没有约朋友相聚，像往日一样读书，刷步，听博客，写blog，陪孩子们弹钢琴，学习，晚上10多点钟就上床睡觉了，不再等跨年的那一刻。

今天又找出了Derek Sivers的文章 Time is personal, your year changes when your life changes 读了读，依旧有被触动到。

A new year begins when there is memorable change in my life, Not January 1st. Nothing changes on the January 1st.

Your year really begins when you move to a new home, start school, quit a job, having a big breakup, have a baby, quit a bad habit, start a new project, or whatever else. Those are the real memorable turning points - where one day is very different than the day before. Those are the meaning markers of time. That is your “new year”.

Celebrate personal meaningful markers, Ignore arbitrary calendar dates.

学到第一招：关于如何精准搜索开源项目

学到第一招：关于如何精准搜索开源项目：即运用高级条件组合选择法（name / readme / description / forks / stars / language / pushed时间 等）

命令如下：

  • 根据名字搜索

​ in:name spring boot stars:>1000 forks:>500

  • 根据Readme文件搜索

​ in:readme spring boot language: java stars:>1000

  • 根据描述搜索

​ in:description spring boot language:java stars:>1000 pushed:>2020-06-01

尝试了一下，果然好用~
