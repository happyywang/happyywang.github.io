成为一名优秀程序员的5个key mindsets

成为一名优秀程序员的5个key mindsets

  1. Be fearless in diving into code you do not know

    This critical skill gets easier as you do it more often. if an open source library or a library from another team does not work as you expect, just jump into the source code to debug why.

  2. Master debugging

  3. Invest in time-saving tools and automating workflows

    building tools - tools for deploying code, for monitoring systems and anything else that enabled them to get more done in less time.

    因为automation enables things to get done in our sleep or while we are focused on other problems.

  4. Optimize your iteration speed

  5. Develop system level thinking

    比如多问自己：

    • How does your code fit in with other parts of the codeabse and features other people are building.

    • Have you sufficiently tested your code?

    • Are customers and users who interact with your code behaving as expected?

    • Will the new code effect the behavior or performance of any other running systems?

Shabbat elevator

Four thousand weeks读到一段有趣且让我小有震撼的情节。书中说纽约东海岸犹太区的一所公寓里的电梯被称为Shabbat elevator. 这个电梯每周会从周五晚到周六晚间这段时间每层楼都会停，即使没有人上下，而原因是这个电梯是专门为这里的犹太住户或访客这样编程的，为了让他们不违反他们的sabbath日禁止操作电子产品的规定。

原来这样的规定里，连电梯里的这个小小的屏幕都被考虑到了。Tim Ferris 在播客里多次提到自己每年都会至少过一个2周grid off的假期。想想自己用电子产品的量和度实在是汗颜。能做点什么改变吗?

Brandon Sanderson对做难事的建议

昨天提到Ryan Holiday说让生活变得更好的方法之一就是就是做相对难的事(Do the more difficult thing)。今天就读到Brandon Sanderson关于做难事的几点建议。

首先他说：I can do hard things. Doing hard things has intrinsic value. and they will make me a better person, even if I end up failing.

接着他给出了3个做Hard things的Tips

Tip #1: Make better goals

好的目标应该是通过行动可触达的，可控的。

Tip #2: Learn how you work

通过实验的方式找出what combination of motivation, and circumstances, and accountability work best for your particular personality.

Tip #3: Break it down

Break large goas down into manageable pieces. for example, create accurate or a more detailed roadmap toward the goals.

什么是兴趣

读到一段关于兴趣的解读：

兴趣是一点点的发掘（discovery）+ 大量的培养（development）+ 一生的深入（deepening)。

兴趣不是通过自省，不是通过询问自己的内心来发现的，而是在不断的尝试探索中发现的。

兴趣光发现还不够，还需要花费大量时间和精力来培养，让它慢慢长大成熟。

外部环境，支持鼓励的家人朋友，对兴趣的发掘和培养，也至关重要。
