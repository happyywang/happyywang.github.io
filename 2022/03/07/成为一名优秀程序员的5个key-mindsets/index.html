成为一名优秀程序员的5个key mindsets

成为一名优秀程序员的5个key mindsets

  1. Be fearless in diving into code you do not know

    This critical skill gets easier as you do it more often. if an open source library or a library from another team does not work as you expect, just jump into the source code to debug why.

  2. Master debugging

  3. Invest in time-saving tools and automating workflows

    building tools - tools for deploying code, for monitoring systems and anything else that enabled them to get more done in less time.

    因为automation enables things to get done in our sleep or while we are focused on other problems.

  4. Optimize your iteration speed

  5. Develop system level thinking

    比如多问自己：

    • How does your code fit in with other parts of the codeabse and features other people are building.

    • Have you sufficiently tested your code?

    • Are customers and users who interact with your code behaving as expected?

    • Will the new code effect the behavior or performance of any other running systems?
