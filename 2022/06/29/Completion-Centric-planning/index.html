Completion Centric planning

Cal Newport的”以完成为中心的计划Completion Centric planning “实施步骤：
Set up: Construct a project page, do the following things:
1. Make an Active Projects List: Professional(work related)/Personal(home, family, fitniss)/Extra(blogging, writing a book, starting a club etc.)
2. Label each project with a completion Criteria
To finish a project you must know “what done looks like”, next, to each project type a concise description of what action must be completed for the project to be completed
3. Label the bottom half of the page as a “Holding Pen”
Using the sysetm: The daily check-in
Each morning, look at your project page and ask: “ What is the most progess I can make toward completing this list today?
Your biggest goal should be to complete projects
If you see a way to do it(even it requires a big push), go for it.
If you can not finish one, think of athe single thing you could do that would get you closest to this goal over the next few days.
Finishing: Rest and reload
Once done, take a break. For at least a week.
