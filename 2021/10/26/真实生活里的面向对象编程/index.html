面向对象编程里的重要元素解读

面向对象编程里的重要元素解读

OOP is a programming paradigm that relies on the concept of classes and objects.

An object is an instance of a class. Class is a blueprint or template from which objects are created.

An object is a real-world entity such as a pen, mobile, keyboard etc. Class is a group of similar objects

  • 封装概念(Encapsulation)

    Encapsulation refers to the bundling of data, along with the methods that operate on the date, into a single unit.

    类比现实世界里的公司，员工是数据，各个部门是不同的方法。

  • 继承概念Inheritance

    Inheritance is the procedure in which one class called the child class inherits the attributes and methods of another class called the parent class.

  • 抽象概念Abstraction

    Abstraction refers to showing only essential and necessary information to the user and hiding unnecessary information.

    类比手机：无论何种手机，都可打电话，拍照，玩游戏，but It does not show you the inside process of how it is doing things, The implementation parts are hidden.

  • 多态概念(Polymorphism)

    Polymorphism refers to many forms. it is the method that performs different things as per the object’s class which calls it.

    there are 2 types of Polymorphism

    1. Static binding or Compile time polymorphism

      method overloading: refers to a class that can have more than one method of the same name but different parameters.

      (类比现实世界里的人，他可以有多重角色，不同角色做不同的事)

    2. Dynamic binding or runtime polymorphism

      method overriding: allows a subclass or child class to provide a specific implementation of a method that is already provided by one of its super-classes or parent classes.
